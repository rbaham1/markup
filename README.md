# Markup

An app that allows a user to add markup lines onto an image.

## Getting Started

Select an image from the list. This will open a picture editing screen.

On the picture editing screen, select a color by clicking on the circle at the bottom of the screen. A color selection pop up will appear. Clicking on a color will select it and return to the picture edit screen.

From there, a user can draw lines on the image using a single finger.

Zoom and pan capabilities are present. Using two fingers, pinch and expand to zoom in and out. To pan, use two fingers and drag in the same direction.

At the top of the picture editing screen, you will see a exit button, a clear button and a save button.
- Exit button will open a confirmation dialog if any markups have been made that are unsaved. This includes clearing previously saved edits. If no unsaved changes have been made, the confirmation dialog will be skipped and return the user to the list of pictures.
- Clear button will remove any markups on the image.
- Save button will save any markups on the image to the picture object so that it can be loaded again later.
